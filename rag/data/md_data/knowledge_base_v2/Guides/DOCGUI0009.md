# Guide : déclarer un sinistre étape par étape

Ce document fait partie de la base de connaissances AssurAuto Maroc, catégorie Guides. Il s'adresse aux conseillers, underwriters et systèmes IA de recommandation.

## Contexte
Ce guide détaille le parcours complet, de la constatation du sinistre jusqu'à la clôture du dossier.

## Détail
Étape 1 : sécuriser les lieux et porter assistance si nécessaire. Étape 2 : remplir le constat amiable. Étape 3 : photographier les dégâts. Étape 4 : déclarer via l'application sous les délais impartis. Étape 5 : suivre l'avancement jusqu'au règlement.

## Application pratique
Étude de cas illustrative : un assuré à El Jadida, avec une Citroën C3 au pack Eco+, applique ce guide étape par étape.

## Points clés
- Sécuriser les lieux avant toute chose
- Remplir le constat le plus complètement possible
- Photographier systématiquement les dégâts
- Suivre le dossier depuis l'espace client

Ce guide a valeur informative générale ; en cas de doute, un conseiller AssurAuto peut être contacté pour un accompagnement personnalisé.
